# Sample Data for Public Demo

This folder contains self-written sample data for the public GitHub version of the StudyMate Agent project.

The original project was built for a personalised multimodal study assistant using course PDFs, OCR text, images, and a user learning profile. However, the original lecture PDFs and private study materials are not included in this public repository due to copyright and privacy considerations.

## Files

- `sample_course_note.txt`: A synthetic course note written for demonstration. It includes example topics such as RAG, OCR, vector databases, agent workflows, PEFT, and LoRA.
- `sample_learning_profile.txt`: A synthetic user learning profile describing preferred explanation style and learning goals.
